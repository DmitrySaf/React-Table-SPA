<h1 align="center">React + Express + PostgreSql application</h1>
<h2>Установка</h2>
Установка с помощью npm: <br>
Шаг 1:

```
git clone https://github.com/DmitrySaf/React-Table-SPA
```

Шаг 2:
Перейти в директорию
  
```
cd React-Table-SPA
```

Шаг 3:
Установить пакеты
  
```
npm i
```

Шаг 4:
Запустить
  
```
npm run app
```
